# recommendation-system-tfidf
Product Recommendation System using Content-Based Filtering and TF-IDF.  Processed 2 million+ supermarket transaction records, performed data cleaning, feature engineering, and built a recommendation engine with 18% error margin.

this project is part of capstone project worked on MSIB program held by startup campus, in this project we worked as a team of 5 people and work on this project in span of 2 month
